# Notion clone

Purpose: Clone task template to practice.

Clone to local need add .env **MongoURI**=**yourMongo**

## implements

- Sidebar
- Tooltip
- Breadcrumb
- Task template
  - Calendar view
    - Modify task at center dialog
    - Modify task at full page
  - Table view
    - modify data at table
- Trash Bar
  - delete
  - restore
  - search
- Todo list
- ContextMenu
  - Delete

## DB Document

- Pages
- Tabs
- Tasks

**data dependent**: task -> tab -> page

## Running

start server & client

    yarn dev
